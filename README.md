## Getting Started

### Prerequisites

* [Node.js](https://nodejs.org/en/)

### Installing Dependencies

From your terminal/command prompt run:

```
npm install
```

### Running

From your terminal/command prompt run:

```
npm start
```

This will run the development server on [localhost:3000](http://localhost:3000).
